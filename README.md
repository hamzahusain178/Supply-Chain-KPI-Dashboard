# Supply Chain KPI Dashboard

## Overview
Supply Chain KPI Dashboard built in Excel analysing 100,000+ real e-commerce orders — tracking on-time delivery rate, regional performance, and order value impact on fulfilment time.

## Tools Used
- Microsoft Excel (Pivot Tables, Charts, Dashboard Design)
- Data sourced from Brazilian Olist E-Commerce Dataset

## KPIs Tracked
- *Total Orders:* 112,650
- *On-Time Delivery Rate:* 88.5%
- *Average Delivery Days:* 12.3 days
- *Late Deliveries:* 12,953

## Key Findings
- High value orders (>200 EUR) take 2.5 days longer to deliver
- Northern regions average 29 days vs 8.76 days in São Paulo
- Audio and home comfort categories have highest late delivery rates
